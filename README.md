## TestingSummary - Simple lightweight unit-testing hpp file
# Example:
    int main()
    {
       int a = 5;
       int b = 10;

       TestEqual(6, a);         
       TestLarger(b, a);         
       TestUnEqual(5, b);       
       TestExpression(a == b); 

       return 0;
    }
